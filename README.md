# MyFootballMatch
**Features :**<br>
-Match schedule in past or next match, can choose lague <br>
-Team list, can choose league<br>
-Team description<br>
-Team players<br>
-Player detail<br>
-Favorite match<br>
-Favorite team<br>
Use api [thesportsDB.com](thesportsDB.com)<br><br><br>
